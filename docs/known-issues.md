# Obscribe Beta Known Issues

Use this as the internal known-issues list while inviting early testers.

## Current Boundaries

- Local notes do not automatically migrate into a cloud account after sign-in.
- Collaboration and shared workspaces are not available.
- Public sharing is not available.
- Export/backup is limited and should not be treated as a full recovery system yet.
- AI features require the user to bring their own provider API key.
- AI is explicit/action-based only; Obscribe does not automatically process notes with AI.
- Mobile browser support exists, but the first beta is web/PWA-first, not native mobile.

## Things To Watch During Beta

- Any sign of save delay, lost edits, or confusing save status.
- Account/session issues after email confirmation or password reset.
- Trash/restore edge cases for notebooks, tabs, and pages.
- Large pasted images/files bloating local browser storage.
- Confusion between local-only mode and signed-in cloud mode.
- Empty states where testers do not know what to do next.

## Current Safety Guidance For Testers

- Do not store mission-critical notes yet.
- Keep a copy of anything important outside Obscribe during beta.
- Report any save, sign-in, or restore issue immediately.
