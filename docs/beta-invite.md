# Obscribe Beta Invite Copy

## Short Invite

Hey — I’m opening a tiny early beta for Obscribe, a calm notebook-style workspace for notes, projects, and ideas.

It’s live enough to test the core loop, but still early. Please don’t store mission-critical notes yet.

What I’d love you to test:

- create an account
- create a notebook, tabs, and pages
- write a few real notes
- refresh/sign out/sign in and make sure your work stays
- try search, Quick Capture, Trash/Restore
- tell me where anything feels confusing, slow, or risky

Start here: https://obscribe.com

Send feedback to support@obscribe.com or the message thread where I invited you.

## Slightly Warmer Invite

I’m looking for a few people to beta test Obscribe — a quieter, notebook-style alternative to heavy note apps.

The goal right now is not tons of features. I’m testing whether the core experience feels clear and trustworthy:

1. capture thoughts
2. organize them into notebooks/tabs/pages
3. search and return later
4. recover from mistakes

Please treat it like an early beta: don’t put anything mission-critical in yet, and expect a few rough edges.

If you try it, I’d especially love to know:

- where you hesitated
- whether saving felt trustworthy
- whether the notebook/tab/page structure made sense
- what felt missing from the core loop

Start here: https://obscribe.com
Feedback: support@obscribe.com
